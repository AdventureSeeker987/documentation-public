---
title: "Get settle PnL nonce"
---

<RequestExample>

```shell
curl 'https://api-evm.orderly.org/v1/settle_nonce'
```
</RequestExample>

**Limit: 10 requests per 1 second**

`GET /v1/settle_nonce`

Retrieve a nonce used for requesting a withdrawal on Orderly Network. Each nonce can only be used once.

<ResponseExample>

```json Response
{
    "success":true,
    "data":{
      "settle_nonce": 1
    },
    "timestamp":1685673968302
}
```
</ResponseExample>

**Parameters**

None
