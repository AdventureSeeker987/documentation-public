---
title: "Get Withdrawal Nonce"
---

Limit: 10 requests per 1 second

<RequestExample>
```shell cURL
curl https://api-evm.orderly.org/v1/withdraw_nonce'
```

</RequestExample>
`GET /v1/registration_nonce`

Retrieve a nonce used for requesting a withdrawal on Orderly Network. Each nonce can only be used once.

<ResponseExample>
```json
{
    "success":true,
    "data":{
      "withdraw_nonce": 1
    },
    "timestamp":1685673968302
}
```
</ResponseExample>
**Parameters**

None
