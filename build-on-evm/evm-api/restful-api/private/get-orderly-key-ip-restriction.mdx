</RequestExample>

**Limit 10 requests per 60 seconds**

`GET /v1/client/orderly_key_ip_restriction`

Retrieves the current IP restriction of a particular Orderly key.

<ResponseExample>

```json Response
{
    "success": true,
    "data": {
        "account_id": "0x0f29bfb4c1bc9fea3f3be46bab6d795e22a6272354b136fde05f6b80cfcad546",
        "orderly_key": "ed25519:8tm7dnKYkSc3FzgPuJaw1wztr79eeZpN35nHW5pL5XhX",
        "ip_restriction_list": [
            "218.190.230.163",
            "218.218.218.218"
        ],
        "tag": "some user defined tags" // Optional, could be null if there wasn't a tag
    },
    "timestamp": 1580794149000
}
```
</ResponseExample>

**Parameters**

| Name        | Type      | Required       | Description                                                                       |
| ----------- | --------- | -------------- | --------------------------------------------------------------------------------- |
| orderly_key | string    | Y              | The Orderly Key to query the IP restriction list                                  |
