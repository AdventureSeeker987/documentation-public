<RequestExample>

```shell cURL
curl -X POST 'https://api-evm.orderly.org/v1/client/reset_orderly_key_ip_restriction' \
    -H 'Content-Type: application/json' \
    -H 'orderly-timestamp: 1649920583000' \
    -H 'orderly-account-id: 0x0f29bfb4c1bc9fea3f3be46bab6d795e22a6272354b136fde05f6b80cfcad546' \
    -H 'orderly-key: ed25519:8tm7dnKYkSc3FzgPuJaw1wztr79eeZpN35nHW5pL5XhX' \
    -H 'orderly-signature: JRaXrOez2OeM20AbpqANI3Rw651BV9c0vBYfvmn9P6XT3jiwWltIKOBNvqEIk+73lnUnZUYyv2m+M1KIQWmPAA==' \
    -d '{"orderly_key": "ed25519:8tm7dnKYkSc3FzgPuJaw1wztr79eeZpN35nHW5pL5XhX", "reset_mode": "ALLOW_ALL_IPS"}'
```
</RequestExample>

**Limit: 10 requests per 60 seconds**

`POST /v1/client/reset_orderly_key_ip_restriction`

Reset the IP restriction of a particular Orderly key under the account.

<ResponseExample>

```json Response
{
    "success": true,
    "timestamp": 1580794149000
}
```
</ResponseExample>
**Parameters**

| Name               | Type    | Required | Description                                                                           |
| ------------------ | ------- | -------- | --------------------------------------------------------------------------------------|
| orderly_key        | string  | Y        | The Orderly Key to set the IP restriction list                                        |
| reset_mode         | string  | Y        | The new mode of the IP restriction for this Orderly Key (ALLOW_ALL_IPS, DISALLOW_ALL_IPS). |
