<RequestExample>

```shell cURL
curl -X POST 'https://api-evm.orderly.org/v1/client/set_orderly_key_ip_restriction' \
    -H 'Content-Type: application/json' \
    -H 'orderly-timestamp: 1649920583000' \
    -H 'orderly-account-id: 0x0f29bfb4c1bc9fea3f3be46bab6d795e22a6272354b136fde05f6b80cfcad546' \
    -H 'orderly-key: ed25519:8tm7dnKYkSc3FzgPuJaw1wztr79eeZpN35nHW5pL5XhX' \
    -H 'orderly-signature: JRaXrOez2OeM20AbpqANI3Rw651BV9c0vBYfvmn9P6XT3jiwWltIKOBNvqEIk+73lnUnZUYyv2m+M1KIQWmPAA==' \
    -d '{"orderly_key": "ed25519:8tm7dnKYkSc3FzgPuJaw1wztr79eeZpN35nHW5pL5XhX", "ip_restriction_list": "10.1.1.1,10.1.1.2,192.168.0.1-192.168.0.10"}'
```
</RequestExample>

**Limit: 10 requests per 60 seconds**

`POST /v1/client/set_orderly_key_ip_restriction`

Set the IP restrictions of a particular Orderly key under the account.

<ResponseExample>

```json Response
{
    "success": true,
    "data": {
        "ip_restriction_list": [
            "10.1.1.1",
            "10.1.1.2",
            "192.168.0.1-192.168.0.10"
        ]
    },
    "timestamp": 1580794149000
}
```
</ResponseExample>

**Parameters**

| Name                | Type    | Required | Description                                                                  |
| ------------------- | ------- | -------- | -----------------------------------------------------------------------------|
| orderly_key         | string  | Y        | The Orderly Key to set the IP restriction list                               |
| ip_restriction_list | string  | Y        | List of IP or IP ranges (comma separated), that will be allowed to place orders with the orderly_key.  |
